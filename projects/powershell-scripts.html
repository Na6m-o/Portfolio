01 / CONTEXTE

Pourquoi le scripting PowerShell ?

En tant que technicien IT, je fais face chaque jour a des taches repetitives : creation de comptes, reinitialisation de mots de passe, rapports d'inventaire, verification de l'etat des services. Faire ca a la main prend du temps — et le temps d'un admin est precieux.

PowerShell s'est impose comme l'outil incontournable de l'administration Windows en entreprise. Me former dessus, c'est autant une necessite professionnelle qu'un investissement pour la suite — notamment pour l'automatisation securite.

02 / SCRIPTS DEVELOPPES

Ce que j'ai code

AD

Gestion des utilisateurs AD

Script de creation en masse d'utilisateurs Active Directory a partir d'un fichier CSV. Import automatique des informations (nom, prenom, departement, OU cible), generation du mot de passe temporaire, ajout aux groupes de securite correspondants.

PowerShell
# Import depuis CSV
$users = Import-Csv "users.csv"
foreach ($user in $users) {
    New-ADUser `
      -Name "$($user.Prenom) $($user.Nom)" `
      -SamAccountName $user.Login `
      -Path $user.OU `
      -Enabled $true
}
New-ADUserImport-CsvSet-ADUser

SEC

Audit des comptes inactifs

Identification des comptes utilisateurs AD inactifs depuis plus de X jours (configurable). Le script liste les comptes, peut les desactiver automatiquement ou envoyer un rapport a l'administrateur pour decision manuelle.

PowerShell
# Comptes inactifs depuis 90 jours
$cutoff = (Get-Date).AddDays(-90)
$inactifs = Get-ADUser -Filter * `
  -Properties LastLogonDate |
  Where { $_.LastLogonDate -lt $cutoff }
Get-ADUserLastLogonDateDisable-ADAccount
MON

Surveillance des services critiques

Script de monitoring qui verifie l'etat des services Windows critiques sur les serveurs du parc. Si un service est arrete, tentative de redemarrage automatique + envoi d'une alerte email a l'equipe IT.

PowerShell
$services = @("Spooler","DNS","DHCP")
foreach ($svc in $services) {
    $s = Get-Service -Name $svc
    if ($s.Status -ne "Running") {
        Start-Service $svc
        Send-MailMessage -To "it@..." ...
    }
}
Get-ServiceStart-ServiceSend-MailMessage
03 / APPORTS

Ce que ce projet m'a apporte

PS

Comprendre la logique du pipeline, la manipulation d'objets, les cmdlets AD, la gestion des erreurs (try/catch) et l'ecriture de scripts robustes et reutilisables.

L'audit et le nettoyage regulier des comptes inactifs est une bonne pratique securite fondamentale. Ce projet m'a mis face a la realite des risques lies aux comptes abandonnés.

OPS

Gain de temps reel

Une tache qui prenait 30 minutes manuellement s'execute en quelques secondes. Ca m'a convaincu de l'importance de l'automatisation dans un role d'admin systeme.
